# noise_rssi说明

## 1. 功能说明
- 芯片提供噪声能量检测功能。可检测环境中当前信道的噪声情况。

## 2. 快速入门

- 烧录SDK工程到EVB测试板中，使用拨码开关切换发射端/接收端（OLED屏幕显示TX/RX）。通过拨码开关切换模组收发模式需复位测试板后才生效。
- 接收端提供噪声能量检测显示功能，通过屏幕显示的"E:"观察能量值。
- 发射端可进行陪测。可选择切换SF/BW/FREQ等参数，可选择MODE:D发射单载波测试。

## 3. 接收端接口说明

- rf_get_channel_rssi();

  读取noise_rssi能量值。

- timer6_open_ms(20);

  设置定时器，定时进行noise_rssi能量值检测。

- int8_t cal_rssi(void);

  计算noise_rssi能量平均值，检测RSSI_MAX_NUM次后进行一次计算，并将计算结果显示到屏幕上。
