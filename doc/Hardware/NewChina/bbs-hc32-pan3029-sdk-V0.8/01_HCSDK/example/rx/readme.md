# rx说明

## 1. 功能说明
- 演示芯片连续接收功能。与tx示例搭配使用。

## 2. 快速入门

- 烧录SDK工程到EVB测试板中，使用USB Mini-B 电源线连接电脑，使用电脑串口助手观察结果，串口波特率115200。

- 串口打印：

  Rx : SNR:xxx ,RSSI: xx
  0x00 0x01 0x02 0x03 0x04 0x05 0x06 0x07 0x08 0x09 
  ###Rx cnt xxx##

  其中固定收包内容为0x00 0x01 0x02 0x03 0x04 0x05 0x06 0x07 0x08 0x09 ，cnt值与tx保持一致

## 3.特别注意

- 当芯片配置进入连续接收模式后，就可以连续进行数据接收。不论接收成功还是接收失败，芯片都会在接收结束后继续保持接收状态，除非用户主动退出该状态（切换至standby3模式可退出接收状态）。
